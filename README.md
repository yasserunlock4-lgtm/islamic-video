# المحتوى الإسلامي - Islamic Video Generator

موقع إنشاء فيديوهات قرآنية حقيقية مع صوت القراء.

## الخطوات - كيفية الرفع على GitHub و Vercel

### 1. تحميل Node.js
- اذهب إلى https://nodejs.org
- حمّل النسخة LTS وثبّتها

### 2. رفع الملفات على GitHub
1. افتح GitHub وأنشئ Repository جديد اسمه: `islamic-video`
2. حمّل كل الملفات الموجودة في هذا المجلد إلى الـ Repository
   (اسحبها وأفلتها في صفحة GitHub)

### 3. ربط Vercel بـ GitHub
1. اذهب إلى https://vercel.com
2. اضغط "New Project"
3. اختر الـ Repository اللي أنشأته `islamic-video`
4. اضغط Deploy
5. انتظر دقيقة واحدة وسيعطيك رابطاً للموقع

### 4. تجربة الموقع محلياً (اختياري)
افتح Command Prompt في مجلد المشروع واكتب:
```
npm install
npm start
```
ثم افتح المتصفح على: http://localhost:3000

## الميزات
- رفع صورة خلفية
- اختيار القارئ من 8 قراء مشهورين
- تحديد السورة وعدد الآيات
- إنشاء فيديو MP4 حقيقي
- تحميل الفيديو مباشرة
- جدول نشر أسبوعي
